# emacs-config

Simple emacs config.

## Supported:
- C/C++
- Rust
- Scheme (MIT Scheme)
- Common Lisp (SBCL)
- Markdown

## Usage:
In your shell:  
- `cd ~`
- `git clone https://github.com/rn7s2/emacs-config.git`
- `ln -s ~/emacs-config/emacs.el ~/.emacs`
Then open emacs,  
- `M-x package-refresh-contents`
- `M-x package-install-selected`  
All done. Don't forget to restart emacs.
